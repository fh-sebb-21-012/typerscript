Welcome to TyperScript

Flex your typing skills with TyperScript - a typing test application written in TypeScript

Play

tl;dr

TyperScript is a typing test application, created for the Web Development course at FH Hagenberg, Upper Austria, made to demonstrate the TypeScript programming language.

With TyperScript, you can measure your typing speed and save your highscores to the leaderboard.

Aside from being programmed in TypeScript, TyperScript is in no way affiliated with the TypeScript project itself!
